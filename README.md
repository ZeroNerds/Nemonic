
## Nemonic


Another browser based around Electron, Zeronet, Beaker, Ory Editor, etc.


### Why?

- January 17th 2021 is fast approaching. :)
- Where's the Cyberpucked 3D Web we were hoping for?
- VR HUDs and gloves are nearly here.
